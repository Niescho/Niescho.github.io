什麼是TensorFlow？

（Wiki）TensorFlow is a free and open-source software library for machine learning and artificial intelligence. It can be used across a range of tasks but has a particular focus on training and inference of deep neural networks.

TensorFlow是一個開源機器學習平台，developed by Google，是目前最受歡迎的平台之一。你可以稱他為“張量流”。為什麼稱為張量？在接下來的過程中你將會逐漸看到。

一個簡單的例子

下面我有一張表，你能看出它們之間有什麼關係嗎？

X -1 0 1 2 3 4
Y -3 -1 1 3 5 7

沒錯是$Y =2X-1$，這種找規律的題目我們小學時便已熟知了，但是交給計算機，計算機怎麼去辨別他們之間有什麼關係呢？

下面我們給出如下程序：

第一步，導入模組tensorflow和numpy，習慣簡稱為tf和np。

第二步，利用tf.keras.Sequential，kears is a high-level API to build and train models in TensorFlow. Sequential model is appropriate for a plain stack of layers where each layer has exactly one input tensor and one output tensor.

当計算機嘗試獲取這些內容时，神經網絡模型会進行猜測，例如是 Y=2X+1。loss 函數將猜測答案与已知正確答案進行對比，並衡量結果的好坏。

接下来，模型使用 optimizer 函數進行另一个猜測。它会根據損失函數的結果，最大限度地減少損失。此时，可能会得到 Y=3X-5 之類的內容。虽然这样仍有些糟糕，但更接近正確的結果（損失更低）。

第三步，numpy將創建兩個ndarry對象，來存儲x和y的值

第四步，訓練神經網絡。训练神经网络的过程是在 model.fit 调用中的，神经网络从其中了解 X 和 Y 之间的关系。在这种情况下，它会在进行猜测、衡量好坏（损失），或通过优化器做出另一个猜测之前遍历整个循环。它会按照您指定的周期数运行。运行该代码时，您会看到每个周期输出的损失。
