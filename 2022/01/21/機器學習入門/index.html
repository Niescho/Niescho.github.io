Part 1：Begining

什麼是機器學習？

對一般的程式來講，我們將規則(Rule)和數據(Data)輸入進電腦中，程式會計算出新的數據。

而對於機器學習（簡單的來說），我們將數據(Data)和結果(Answer)輸入進電腦中，程式會推測出這個“遊戲”的規則(Rule)是什麼（當然結果不會是百分百準確）。

​

之後我們便可以輸入數據(Data)並給出預測(Prediction)

比如說AlphaGo，運用了大量的機器學習技術，其繼任者AlphaZero只需要和自己下棋，就可以打敗前任。

（Wiki）Machine learning (ML) is the study of computer algorithms that can improve automatically through experience and by the use of data.[1] It is seen as a part of artificial intelligence. Machine learning algorithms build a model based on sample data, known as training data, in order to make predictions or decisions without being explicitly programmed to do so.[2]

Mechine learning的approch主要有三類：

  1. Supervised learning 監督學習：The computer is presented with example inputs and their desired outputs, given by a “teacher”, and the goal is to learn a general rule that maps inputs to outputs.
  2. Unsupervised learning 非監督學習：No labels are given to the learning algorithm, leaving it on its own to find structure in its input. Unsupervised learning can be a goal in itself (discovering hidden patterns in data) or a means towards an end (feature learning).
  3. Reinforcement learning 強化學習：A computer program interacts with a dynamic environment in which it must perform a certain goal (such as driving a vehicle or playing a game against an opponent). As it navigates its problem space, the program is provided feedback that’s analogous to rewards, which it tries to maximize.[4]

什麼是TensorFlow？

（Wiki）TensorFlow is a free and open-source software library for machine learning and artificial intelligence. It can be used across a range of tasks but has a particular focus on training and inference of deep neural networks.

TensorFlow是一個開源機器學習平台，developed by Google，是目前最受歡迎的平台之一（另一個是Pytorch，developed by Facebook)。你可以稱他為“張量流”。張量其實就是一張$n\times n$的表格。為什麼稱為張量？在接下來學習的過程中你將會逐漸看到。

下面我有一張表，你能看出它們之間有什麼關係嗎？

下面我們給出如下程序：

第一步，導入模組tensorflow和numpy，習慣簡稱為tf和np。

第二步，利用tf.keras.Sequential來構建神經網絡，kears is a high-level API to build and train models in TensorFlow. Sequential model is appropriate for a plain stack of layers where each layer has exactly one input tensor and one output tensor.

当計算機嘗試獲取這些內容时，神經網絡模型会進行猜測，例如是 Y=2X+1。loss 函數將猜測答案与已知正確答案進行對比，並衡量結果的好坏。

接下来，模型使用 optimizer 函數進行另一个猜測。它会根據損失函數的結果，最大限度地減少損失。此时，可能会得到 Y=3X-5 之類的內容。虽然这样仍有些糟糕，但更接近正確的結果（損失更低）。

第三步，用numpy創建兩個ndarry對象，來存儲x和y的值

第四步，**訓練(train)**神經網絡。訓練神经网络的过程是在 model.fit 调用中的，神经网络从其中了解 X 和 Y 之间的关系。在这种情况下，它会在进行猜测、衡量好坏（损失），或通过优化器做出另一个猜测之前遍历整个循环。它会按照您指定的周期数运行。运行该代码时，您会看到每个周期输出的损失。

该网络的第一层 tf.keras.layers.Flatten 将图像格式从二维数组（28 x 28 像素）转换成一维数组（28 x 28 = 784 像素）。将该层视为图像中未堆叠的像素行并将其排列起来。该层没有要学习的参数，它只会重新格式化数据。

展平像素后，网络会包括两个 tf.keras.layers.Dense 层的序列。它们是密集连接或全连接神经层。第一个 Dense 层有 128 个节点（或神经元）。第二个（也是最后一个）层会返回一个长度为 10 的 logits 数组。每个节点都包含一个得分，用来表示当前图像属于 10 个类中的哪一类。

在Dense中我們有新的參數activation(激勵函數):

  • relu（Rectified Linear Uni）线性整流函数（将输入小于0的值幅值为0，输入大于0的值不变）。
  • softmax归一化,我們了一個長度為10的數組，該函數將它們之中最大的值設為1.0，其他值都設為0。

新的參數metrics- 用于监控训练和测试步骤，你將會在每一個步驟看到它的值。在這裏使用了accuracy，即被正确分类的图像的比率。
